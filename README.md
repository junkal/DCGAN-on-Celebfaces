# DCGAN on Celebrity Face data set

The script is an exercise of following a tutorial to train a GAN using Celebrity Faces' data set to be able to generate new faces. The data set is found from [Kaggle](https://www.kaggle.com/jessicali9530/celeba-dataset)

![image](https://user-images.githubusercontent.com/6497242/137588860-6c313574-1545-4202-92f8-fc3b7aeefed1.png)

The outputs generated from the GAN is face from being realistic like those generated using StyleGAN. But they nonetheless do look like faces
